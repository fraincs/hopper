# Library

All available rich icons in the icon library are shown below.

Use the correct size for each icon. Icons should be used at their original size according to the design.

The color of the rich icons can be changed by using the `variant` prop. 8 Decorative colors are available for the rich icons.
```tsx
<ActionListRichIcon variant="option3" />
```

Click or tap on any icon to copy its name.
